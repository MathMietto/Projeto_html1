História do HTML

(Voltar)

Criado em 1991 Tim Berners-Lee, na Suíça, o HTML (HyperText Markup Language), foi incialmente projetado para interligar instituições de pesquisas próximas, e também para realizar o compartilhamento de documentos com facilidade.

Em 1992, foi liberada a biblioteca de desenvolvimento WWW (World Wide Web) , que junto com o HTML realizou um alcance mundial de uso da web.

O HTML é usada com a finalidade de realizar marcações de texto, sendo assim não é considerado uma linguagem de programação e sim de texto.

Estrutura

(Voltar)

A estrutura do HTML é divida em 3 pontos. Sendo eles:

  1. Html
  2. Head
  3. Body

Html - É a tag mãe, pois é a partir dela que iniciamos uma estrutura em HTML.

Head - Esta tag utilizamos ela para inserir todas as informações que serão pré carregadas antes de serem exibidas ao usuário.

Body - Dentro desta tag iserimos todo o conteúdo que será exibido ao usuário ao acessar a página, como, imagens, textos, etc...

Algumas tags

(Voltar)

Abaixo foram listadas algumas tags que são utilizadas no HTML e o que elas são capazes de fazer:

h1 - Esta tag é utilizada para um título inicial em uma página, o título que possui mais peso

a tag h1 varia dentre (h1 até o h6), isso também em sua relevancia, sendo o h2 um subtítulo e assim sucessivamente

p - Esta tag é utilizada para paragáfos em uma página dentro do HTML. É com esta tag que inserimos textos.

strong - Esta tag é utilizada para colocar palavras em NEGRITO.

ul - Utilizada para criar listas não ordenadas.

ol - Utilizada para criar listas ordenadas.

As listas tanto ordenadas quanto não ordenadas é necessário utilizar a tag li para cada item inserido nas listas.

As tags possuem um padão de abertura < p > e fechamento < / p >. Exemplo:

< p > Algum parágrafo aqui < / p >

.

Por que o HTML é importante?

(Voltar)

O HTML é a base da web! Pois ele concede a possibilidade da estrutura básica da web.

Ele auxilia no desenvolvimento web, organizando as páginas e dando significado as mesmas que acessamos no dia a dia.
